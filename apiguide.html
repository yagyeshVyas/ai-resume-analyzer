🔧 Error Codes & How to Fix Them
401 Unauthorized
🔴 Cause: Wrong API key, extra spaces, or expired key
✅ Fix: Copy key again carefully. No spaces before/after. Check provider dashboard.
404 Not Found
🔴 Cause: Model was removed, renamed, or wrong model ID
✅ Fix: Switch to Auto Free Router (OpenRouter) — it never 404s. Or try a different model.
429 Rate Limit
🟡 Cause: Too many requests (hit free tier limit)
✅ Fix: Wait 1 minute. Or switch to a different provider/model. Or add credits.
402 Payment
🔴 Cause: Out of credits on paid provider
✅ Fix: Add credits at provider dashboard, or switch to a free provider.
503 Service Unavailable
🟡 Cause: Model is loading (HuggingFace cold start) or provider down
✅ Fix: Wait 30–60 seconds, then retry. Normal for first HuggingFace request.
Timeout
🟡 Cause: Slow/overloaded model taking too long
✅ Fix: Switch to Groq (fastest), or OpenRouter Auto Router.
Invalid JSON
🟡 Cause: Model returned garbled output (usually small/weak models)
✅ Fix: Switch to a larger model like Llama 3.3 70B or Claude 3.5 Sonnet.
🚨 OpenRouter-specific: If you get a 404 on a free model, it may have been removed. Always use 🎲 Auto Free Router (openrouter/free) as it self-updates and can never 404.
